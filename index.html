Overview

Creating a large-scale hotel-booking application has always been one of my objectives. This project is a result of my determination and hardwork.

Project Difficulties

There were a number of challenges in developing this app. Some came from changes with the latest release.

I meant this to be a large scale application so I spent time planning and researching extensively to avoid huge adjustments along the way.

Some routing with wildcard issues that had strange debugging error were encountered.

My Solution

Laravel is an elegant framework and through its documentation and the abundance of help from community, I resolved a lot of challenges like errors occuring during migrations from older laravel version, and 404 error after making changes on routes just to name a few.

Coming from Laravel v5.7, I found that the previous migrations auto increment "increments(id)" was replaced by "id()" which implemented a data type change from Integer into Big Integer. It was an accomplishment because the debugger did not specify the data type error.

Repetitive tasks were made easy through its Eloquent queries and I took advantage of it. It was a delight to use and I'm thrilled to make another project again.
